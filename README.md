## Description
`cub3d` is a 3D graphical representation of the inside of a maze from a first-person perspective, created using ray-casting principles (similar to the classic game Wolfenstein 3D).

## Instructions
**Compilation:**
Run `make` to compile the `cub3D` executable.

**Execution:**
Start the program with a scene description map file.
```
./cub3D maps/good/test.cub
```

**Controls:**
- **W, A, S, D**: Move around the map.
- **Left/Right Arrows**: Look around.
- **ESC**: Exit the game.
